## Variant: Kanban Ops Command Center

### Design stance
Una vista tipo centro de mando: columnas operativas, métricas arriba y una barra lateral de prioridades.

### Key choices
- Layout: Kanban horizontal con sidebar sticky.
- Typography: fuerte y compacta para lectura rápida.
- Color: dark/NexCard emerald con alertas ámbar/rojo.
- Interaction: botón de nota muestra feedback inmediato.

### Trade-offs
- Strong at: operación diaria con alto volumen y lectura de prioridades.
- Weak at: requiere ancho de pantalla; en mobile se vuelve scroll horizontal.

### Best for
- Operador interno que revisa pedidos varias veces al día y necesita saber la próxima acción.
