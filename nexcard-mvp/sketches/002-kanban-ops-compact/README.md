## Variant: Kanban Ops Compacta

### Design stance
Una bandeja de trabajo compacta, enfocada en velocidad y mínima distracción.

### Key choices
- Layout: sidebar de navegación + tablero con columnas compactas.
- Typography: números y acciones visibles, menos narrativa.
- Color: dark sobrio con verde para progreso y rojo/ámbar para bloqueo.
- Interaction: drawer de detalle rápido para no salir del flujo.

### Trade-offs
- Strong at: operación repetitiva diaria, menos ruido visual, más velocidad.
- Weak at: menos “wow”; requiere que el equipo ya entienda el proceso.

### Best for
- Uso real de backoffice, especialmente cuando una persona revisa varias órdenes seguidas.
